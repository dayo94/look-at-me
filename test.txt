

깃 올리는 순서

파일을 수정 및 생성 후

git status 

수정된 파일, 추가된 파일 상태를 보기
(가끔 깨지는 파일이 있으니 untrack 같은 키워드가 뜨면 구글링 ㄱㄱ)

git add . 
-> status에 올라온 파일 전체 깃에 올리기

git add (올리고 싶은 파일명 or 폴더명) - 하나만 올리고싶을땐 . 절대 쓰지마 ㅎㅎ

git status
add한 파일이 제대로 올라갔는지
(녹색이면 제대로 올라간 파일)

git commit -m "~~~수정"
(따움표 안에 꼭 적어줘)

git push origin master
